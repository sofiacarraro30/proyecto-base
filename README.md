## Maquetación de proyecto web

# Clon de Whatsapp - Entrega 


## Descripción
En este proyecto encontrarás una maquetación en HTML y CSS de una interfaz tipo Whatsapp. La maqueta esta conformada por un sidebar, donde podras visualizar los chats; y un panel principal con la conversación de uno de los usuarios. Las tecnologías utilizadas en este proyecto fueron HTML y CSS. 

## Estructura del proyecto
El proyecto esta estructurado de la siguiente manera. 

/index.html
styles/main.css
./assets/ (esta carpeta contiene las imagenes utilizadas en el proyecto)

## Visualización del proyecto
Para poder visualizar el proyecto podes abrir index.html en cualquier navegador. 

## Decisiones de diseño
La paleta cromatica elegida en este proyecto es la siguiente. Esta misma corresponde a colores similares a los de Whatsapp.

--primary-color: #0a9c6f;
  --grey-900: #0d1919;
  --grey-800: #121b22;
  --grey-600: #1f2c34;
  --grey-700: #9f9fa1;
  --grey-500: #dfdfe1;
  --grey-300: #fefff6;
  --grey-200: #f1f2f6;
  --green-400: #138b80;
  --green-300: #2dca7a;
  --green-100: #dcf8c6;

  La tipografía utilizada fue "Reboto" enlazada mediante Google Fonts. 
  Utilizamos Grid para poder dividir la pantalla en dos columnas, la del sidebar y la del panel de conversación. Esto nos permitio que sea mucho mas facil el armado. También utilizamos Flexbox. Esto permitio que algunos de los componentes puedan estar alineados de manera ordenada y visualmente armonica. 
  Los estilos fueron organizados con clases para poder identificarlos y ordenarlos en la hoja.
